# 漢字 Lexicon

A static web tool for generating Japanese vocabulary and Kanji tables directly from authentic dictionaries.

## Features
- **100% Client-Side**: Hosted seamlessly on GitHub Pages without any backend servers.
- **Native Dictionaries**: Securely cross-references `kanjiapi.dev`, `Jisho.org` (via public proxy), and `ja.wiktionary.org`.
- **Export Ready**: Download your generated tables directly locally as `.csv` or `.xlsx` without any backend processing.
- **Minimalist Aesthetic**: Features Japanese Mincho typography layout respecting system dark-mode.
